# Presenter notes - Session 1: Meet Codex

## Before the session
- Test the installer on a CLEAN machine (or fresh user account) the day before. The curl script, npm, and brew paths all behave differently behind corporate proxies - know which one works in your venue's network.
- Have the desktop-app path rehearsed and ready for terminal-averse attendees: install ChatGPT app, open folder, pick Codex mode. You will need it within the first 10 minutes, guaranteed.
- Confirm your own Plus account is signed in and has window headroom - do not burn your 5-hr window on morning-of testing.
- Projector check: terminal font at 18pt+, dark theme, zoom toggle on the course page works.

## Timing (45 min)
- 0-3: Welcome, why this course exists, the one-website promise
- 3-9: Part 1 - agent vs autocomplete, five surfaces
- 9-14: Part 2 - setup + permissions concept
- 14-18: Part 3 - GCOB structure
- 18-26: Demo 1 - install, sign in, first conversation (everyone)
- 26-36: Demo 2 - plan YOUR site with GCOB (everyone)
- 36-40: Demo 3 - feel the permission gate
- 40-45: Homework walkthrough + Q&A

## Demo run-of-show
- Demo 1 key beat: the "introduce yourself" first conversation. Let Codex's own answer do the feature tour - resist narrating over it.
- Demo 2 key beat: EVERYONE leaves with a SITE_PLAN.md file. This is the session's contract - it gates Session 2. Never cut the revision lap where they disagree with the plan out loud; that is where ownership starts.
- Demo 3: keep it light. The point is feeling the approval prompt, not permissions theory.

## Watch for
- Install failures cluster in the first 5 minutes of Demo 1. Pair stuck people with a neighbor immediately - fix machines after, not during.
- Terminal-averse attendees freeze silently. Offer the desktop app path early and without ceremony.
- People who write step-by-step prompts instead of GCOB. Redirect gently: "what should exist when it's done?"

## Cuts if long
- Cut Demo 3 entirely (permission gate resurfaces naturally in Session 2).
- Compress Part 1's five-surfaces table to one sentence: "five doors, one agent, we live in the CLI."
- Models/credits detail is already self-study - never present it live.
