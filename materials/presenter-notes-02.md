# Presenter notes - Session 2: First build

## Before the session
- Build your OWN my-site one full session ahead - you need a working homepage of your own to show, and you need to have hit the same errors attendees will hit.
- Verify everyone's SITE_PLAN.md homework landed: message the group two days ahead; have a 3-line fallback plan file ready for anyone who arrives without one.
- Pre-run /init on a scratch copy so you know what today's AGENTS.md output looks like - it changes between Codex versions.
- Have a browser window pre-positioned next to the terminal for the refresh-and-see loop.

## Timing (45 min)
- 0-3: Welcome back, SITE_PLAN.md roll call
- 3-10: /init and what AGENTS.md is (concept level - depth is Session 3)
- 10-22: Demo 1 - homepage from the plan (build-along)
- 22-34: Demo 2 - three iteration laps (the heart of the session)
- 34-41: Demo 3 - break it and let Codex fix it
- 41-45: Homework + Q&A

## Demo run-of-show
- Demo 1: first real build. Let the plan file drive - "build the homepage from SITE_PLAN.md" - so they see their OWN plan become code.
- Demo 2 key beat: the three iteration laps - NEVER cut these. Lap 1 look, lap 2 taste ("more whitespace, smaller hero"), lap 3 content tweak. Iteration is the actual skill of this course; the homepage is a side effect.
- Demo 3: sabotage something visible (delete a closing tag), watch it break, paste the symptom to Codex. The fix lands fast - the lesson is "describe symptoms, not diagnoses."

## Watch for
- People who accept lap 1 output as final. Push them: "what would make it feel more like YOU?" Nobody's first render is their taste.
- The browser-refresh rhythm is not obvious to non-engineers - demonstrate the save/refresh/react loop explicitly, twice.
- Divergent plans (someone's plan wants a shop, a login...) - park scope kindly: "session 6 you can build anything; today, homepage."

## Cuts if long
- Cut Demo 3 (break-and-fix) - it reappears organically the first time anything breaks in Session 4.
- One iteration lap can go to homework, but never below two live laps.
- AGENTS.md explanation can shrink to 90 seconds; Session 3 owns it anyway.
